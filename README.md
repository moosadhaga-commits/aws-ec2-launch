# # AWS EC2 Launch Project

## Objective
Launch an EC2 instance on AWS.

## Services Used
- Amazon EC2
- Security Group
- Key Pair

## Steps
1. Logged in to AWS Console.
2. Launched an EC2 instance.
3. Selected Amazon Linux 2 AMI.
4. Created a Security Group.
5. Connected to the instance using SSH.

## Skills Learned
- EC2
- Security Groups
- SSH
